Generating predictive alerts for customer events

Cloud Implementation

I implemented the entire project as a scalable and fully orchestrated end-to-end MLOps pipeline built using the AWS Platform.

The AWS MLOps solution is perfectly coherent, with minimal manual interventions and no redundant tasks. The build includes everything from ETL pipelines to model training, inference, model scoring and deployment with monitoring.

Everything was streamlined and completely automated with bi-weekly schedules to trigger the pipeline, generate alerts, and send across the deliverables to the stakeholder. The entire process is logged with an email notification notifying the pipeline status.
